# Cara Membuat Akun GitHub

GitHub adalah platform untuk menyimpan dan mengelola kode (repository) berbasis Git. Berikut langkah-langkah membuat akun GitHub dari awal.

## Step 1: Buka Website GitHub
1. Buka browser (Chrome, Firefox, dll).
2. Akses situs resmi GitHub:
https://github.com/

## Step 2: Klik Tombol Sign Up
1. Di halaman utama GitHub, klik tombol **Sign up**.
2. Biasanya tombol ini ada di pojok kanan atas atau di tengah halaman.

## Step 3: Isi Data Pendaftaran
Isi data yang diminta oleh GitHub:
- **Email address**: masukkan email aktif
- **Password**: buat password yang kuat
- **Username**: pilih nama pengguna (username) yang unik

Setelah itu klik **Continue**.

## Step 4: Verifikasi Akun
1. GitHub akan mengirim kode verifikasi ke email.
2. Buka email lu dan masukkan kode tersebut ke halaman GitHub.

## Step 5: Pilih Preferensi (Opsional)
GitHub akan menanyakan beberapa pertanyaan seperti:
- Tujuan menggunakan GitHub
- Tingkat pengalaman coding

Bagian ini bisa diisi atau dilewati.
## Step 6: Pilih Paket
1. Pilih **Free** (gratis).
2. Klik **Continue for free**.

## Step 7: Akun GitHub Berhasil Dibuat
Jika semua langkah selesai, akun GitHub lu sudah aktif dan siap digunakan.

## Penutup
Sekarang lu sudah punya akun GitHub dan bisa:
- Membuat repository
- Upload project
- Belajar Git dan kolaborasi dengan orang lain
