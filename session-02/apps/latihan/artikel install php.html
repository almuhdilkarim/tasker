Tutorial Instalasi PHP di Windows

Pada panduan ini, kita akan mempelajari cara menginstal PHP di sistem operasi Windows secara manual tanpa menggunakan XAMPP.

Langkah 1: Unduh PHP

Kunjungi situs resmi PHP:

https://windows.php.net/download/

Pilih versi terbaru (misalnya PHP 8.3) dan unduh versi Thread Safe untuk sistem Anda (x64 atau x86).

Langkah 2: Ekstrak File PHP

Setelah diunduh, ekstrak file ZIP ke folder, misalnya:

C:\php

Langkah 3: Tambahkan PHP ke PATH Environment Variable

Buka System Properties → Advanced → Environment Variables, lalu tambahkan path berikut ke variabel Path:

C:\php

Kemudian klik OK untuk menyimpan perubahan.

Langkah 4: Cek Instalasi PHP

Buka Command Prompt dan ketik perintah berikut:

php -v

Jika instalasi berhasil, akan muncul output seperti ini:

PHP 8.3.0 (cli) (built: Nov 21 2024 10:00:00) (ZTS Visual C++ 2019 x64)
        Copyright (c) The PHP Group
        Zend Engine v4.3.0, Copyright (c) Zend Technologies

Langkah 5: Uji PHP di Browser

Buat file baru bernama info.php di dalam folder C:\xampp\htdocs (atau folder web server Anda), lalu isi dengan kode berikut:

<?php
    phpinfo();
    ?>

Simpan file tersebut, lalu buka di browser dengan alamat:

http://localhost/info.php

Jika halaman udah nampilin informasi versi PHP, berarti installnya sudah sukses
